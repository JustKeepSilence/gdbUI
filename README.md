# gdbui
=

#### Build Setup

``` bash
# install dependencies
npm install

# install electron
npm install electron@8.3.1
npm rebuild --target=8.3.1 --runtime=electron --dist-url=https://atom.io/download/electron

# build electron application for production
npm run build

# dev

npm run dev
```
